1. Variables

Variables in Python are used to store data values. A variable is created by assigning a value to a name. For example:

# Creating a variable in Python
x = 5
name = "John"
print(x)
print(name)

Here, x is a variable storing the integer value 5, and name is a variable storing the string "John".

2. Operators

Operators are used to perform operations on variables and values.
In the example below we use the '+' operator to add two integer values together. Example:

print(10 + 5)

Python groups operators within the following categories:

  • Arithmetic operators
  • Assignment operators
  • Comparison operators
  • Logical operators
  • Identity operators
  • Membership operators
  • Bitwise operators

3. Data Types

Python has various data types, including integers, strings, floats, and lists. Here's an example:

# Integer
x = 10
# String
name = "Alice"
# Float
height = 5.6
# List
fruits = ["apple", "banana", "cherry"]

In Python, you can easily assign values of different types to variables.

4. Loops

Loops are used to repeat a block of code multiple times. Python provides for and while loops. Example of a for loop:

# For loop example
for i in range(5):
   print(i)

This code prints numbers from 0 to 4.

5. Functions

Functions in Python allow you to group code into reusable blocks. You can define a function using the def keyword:

# Function definition
def greet(name):
   return "Hello, " + name

# Function call
result = greet("Alice")
print(result)

This function takes an input (name) and returns a greeting message.

6. Classes

In Python, classes are used to define objects. A class is like a blueprint for creating objects.
The __init__ function is an initialization function that is executed each time a Person object is created, in this example when a Person object is created the name and age of the object is assigned to the values passed when creating the object.
Example:

# Class definition
class Person:
   def __init__(self, name, age):
      self.name = name
      self.age = age

# Creating an object
person1 = Person("John", 30)
print(person1.name, person1.age)

Here, a Person class is defined, and an object is created with specific attributes (name and age).

7. User Input

Python allows for user input. That means we are able to ask the user for input.
The following example asks for the username, and when you entered the username, it gets printed on the screen:

username = input("Enter username:")
print("Username is: " + username)
