Python Setup Guide

Follow these steps to set up Python on your machine:

Step 1: Download Python

Visit the official Python website and download the latest version:

  • Download Python

Step 2: Install Python

After downloading the installer, follow these instructions:

  • Run the Python installer and make sure to check the box that says “Add Python to PATH”.
  • Click “Install Now” and follow the installation prompts.

Step 3: Verify Installation

Once the installation is complete, open your terminal or command prompt and type the following command:

python --version

If Python is correctly installed, it will display the version number. For example:

Python 3.x.x

Step 4: Install pip (if not installed)

If pip (Python package installer) is not installed by default, you can install it manually. Run the following command:

python -m ensurepip --upgrade

Step 5: Test Python Setup

To verify everything is working, you can write a simple Python script. Open a Visual studio code and create a file called test.py. Inside, write the following:

print("Hello, Python!")

Then, run the script using this command:

python test.py

If you see the output Hello, Python!, everything is set up correctly.
