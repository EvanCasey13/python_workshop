Packages

Packages in python allow for additional code libraries and functionality to be easily implemented in any project you have. We will use the Turtle package here as an example.
Turtle graphics is a fun and interactive way to learn programming using Python. It allows you to control a virtual "turtle" (a pen) that moves around the screen, drawing lines, shapes, and patterns based on commands you give it.

# Import turtle package
import turtle

Create window for the turtle to work in

screen = turtle.Screen()

Create a turtle called my_turtle

my_turtle = turtle.Turtle()

Below are examples of different commands you can give your turtle

# Move turtle forward and backwards my_turtle.forward(100) # Move the turtle forward by 100 units my_turtle.backward(50) # Move the turtle backward by 50 units

# Turn turtle left and right my_turtle.left(45) # Turn 45 degrees left my_turtle.right(90) # Turn 90 degrees right

# Function to draw a square def draw_square(): for _ in range(4): my_turtle.forward(100) my_turtle.left(90) draw_square() # Call the function to draw a square

# Include the line below to state the drawing is finished turtle.done()

Here is a full example for testing you can run. First create a python file called turtle.py

import turtle # Set up the screen screen = turtle.Screen() screen.bgcolor("white") # Create a turtle named "star_turtle" star_turtle = turtle.Turtle() star_turtle.shape("turtle") star_turtle.speed(3) # Function to draw a star def draw_star(size): for _ in range(5): # A star has 5 points star_turtle.forward(size) star_turtle.right(144) # Turns the turtle 144 degrees to the right (five times due to the loop) # Draw the star draw_star(100) # Finish turtle.done()
