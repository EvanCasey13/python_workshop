2. Creating a Simple Python API with Flask

A Python API allows us to expose data and functionality via HTTP requests. One popular framework for building APIs in Python is Flask. Here's how to create a simple Python API with Flask:

# Import Flask
from flask import Flask, jsonify

# Initialize Flask app
app = Flask(__name__)

# Sample data to be returned by the API
data = {
"name": "John Doe",
"age": 30,
"city": "New York"
}

# Define a route to expose data
@app.route('/api/data', methods=['GET'])
def get_data():
return jsonify(data)

# Run the app
if __name__ == '__main__':
app.run(debug=True)

In this example:

  • Flask is imported and used to create a web application.
  • jsonify() is used to return the data in JSON format.
  • The /api/data route is defined to handle HTTP GET requests and return the data dictionary.
  • app.run(debug=True) starts the Flask development server with debugging enabled.

To run this API, you need to install Flask first:

pip install flask

Once you have Flask installed, save the Python code in a file (e.g., app.py) and run it using the command:

python app.py

Once the server is running, you can test the API by navigating to http://127.0.0.1:5000/api/data in your browser or using a tool like Postman. The response should look like this:

{
"name": "John Doe",
"age": 30,
"city": "New York"
}
