Introduction to Python Syntax

Python is a versatile, easy-to-learn programming language that emphasizes readability and simplicity. Let's explore the core elements of Python syntax such as variables, loops, functions, and more.

Variables and Data Types

In Python, you don't need to declare the type of a variable. You simply assign a value to it, and Python figures out the type automatically.

# Assigning a number to a variable
x = 10 # Integer
y = 20.5 # Float
name = "Alice" # String
is_valid = True # Boolean

If-Else Statements

Python uses if-else statements to make decisions based on conditions.

# Check if a number is positive, negative, or zero
number = -10
if number > 0:
print("Positive")
elif number < 0:
print("Negative")
else:
print("Zero")

Loops

Loops allow you to repeat a block of code multiple times. Python supports both for and while loops.

# Using a for loop to iterate over a range
for i in range(5):
print(i) # Prints numbers from 0 to 4

# Using a while loop
counter = 0
while counter < 5:
print(counter)
counter += 1 # Increment counter

Functions

Functions are reusable blocks of code. You can define a function using the def keyword.

# Defining a simple function
def greet(name):
print(f"Hello, {name}!")
greet("John") # Calling the function

Classes and Objects

Python is an object-oriented language, and you can define your own classes and create objects based on them.

# Define a simple class
class Dog:
def __init__(self, name, breed):
self.name = name
self.breed = breed
def bark(self):
print(f"{self.name} says Woof!")
# Creating an object from the class
my_dog = Dog("Buddy", "Golden Retriever")
my_dog.bark() # Calling the bark method

Modules and Packages

Python allows you to import and use external code libraries (modules) to add functionality. For example, the math module allows us to work with mathematical operations.

# Importing a module
import math
print(math.sqrt(16)) # Prints the square root of 16

File Input/Output

Python provides easy methods for reading from and writing to files.

# Writing to a file
with open("example.txt", "w") as file:
file.write("Hello, world!")
# Reading from a file
with open("example.txt", "r") as file:
content = file.read()
print(content)
